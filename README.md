# Loja Retrô Games

## Descrição do projeto
Projeto fullstack utilizado para gerenciar produtos de uma loja de videogames retrô, com cadastro, listagem, edição e exclusão de itens.

## Tecnologias

### Aplicativos
- VS Code  
- Navegador  

### Frameworks
- Node.js  
- Express  
- MySQL  
- JavaScript  

## Como executar

Clonar repositório (git clone + url)

Abrir o projeto no VS Code

### Backend
Entrar na pasta Back  
Instalar dependências:
npm install  

Rodar o servidor:
node server.js  

### Frontend
Abrir a pasta Front  
Abrir o arquivo index.html com Live Server

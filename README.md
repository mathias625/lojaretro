# Loja Retrô Games

## Descrição do projeto
Projeto fullstack utilizado para gerenciar produtos de uma loja de videogames retrô, com cadastro, listagem, edição e exclusão de itens.

## Tecnologias

### Aplicativos
- VS Code  
- Navegador  

### Frameworks
- Node.js  
- Express  
- MySQL  
- JavaScript  

## Como executar

Clonar repositório:
git clone URL_DO_REPOSITORIO

Entrar na pasta do projeto:
cd lojaretro

Abrir o projeto no VS Code:
code .

### Backend

Entrar na pasta Back:
cd Back

Instalar dependências:
npm init -y  
npm install express cors mysql2 dotenv  

Rodar o servidor:
node server.js  

### Frontend

Entrar na pasta Front:
cd ../Front

Abrir o projeto:
- Clicar com botão direito no index.html
- Abrir com Live Server

## Testar o projeto

Acessar no navegador:
http://localhost:3000/produtos

O projeto já está pronto para o teste!
